# Parallax Effect Website

This repository contains a simple website created with a parallax effect to enhance the user experience and add depth to the design. The parallax effect creates an illusion of depth by moving different layers of content at different speeds as the user scrolls down the page.

## Features

- **Parallax Effect:** Utilizes the parallax scrolling technique to create a visually engaging experience.
- **Responsive Design:** The website is designed to be responsive, ensuring a seamless experience across various devices and screen sizes.
- **Easy Navigation:** Smooth scrolling and intuitive navigation make it easy for users to explore the content.
- **Customizable:** The code is well-structured and easily customizable, allowing developers to modify and adapt the website according to their needs.

## Technologies Used

- HTML5
- CSS3
